twainmodule
===========

This is the binary distribution of the twain module.

The directory structure is as follows:

.\twain.pyd - the dynamic library implementing the python twain module
.\demo - demonstration sources
.\docs - documentation
.\twexplore - the twexplore program source

To install, extract this zip file into a folder. Copy the 
twain.pyd file to your python site-packages directory, 
i.e. c:\python24\lib\site-packages.
